## 使用说明

## 功能说明

- 使用Spring Data JPA处理表数据，并自动建表
- 配置信息保存在自定义的数据库中,由使用者自己维护数据安全
- 指定数据库(目前仅支持MySQL)进行备份并生成备份文件
- 将指定目录的数据同步到云端(云端存储如FTP、阿里云OSS、华为云OBS、七牛云Kodo、Webdav等)
- 界面使用光年后台模板
